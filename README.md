## **一、项目简介**
表格OCR识别、表格识别模型、表格转Excel、表单识别 支持有线表格识别，支持单元格合并的表格，支持图片方向矫正，可实现单个字符坐标的解析，可将识别结果倒入到Excel中，以及生成html表格。 感兴趣的可以先联系进行体验。
